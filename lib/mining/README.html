go-mining

Go-mining is a small library for data mining.

The library is written in Go language.

Features

Classifiers

  • CART
  • Random Forest
  • Cascaded Random Forest
  • K-Nearest Neighbourhood

Resampling

  • SMOTE
  • LN-SMOTE (Local Neigbourhood SMOTE)

Miscellaneous

  • Gini index
